# ytfam-manager (ytmgr)

_this is a slightly modified fork of my other project [spotifam-manager](https://github.com/userbyte/spotifam-manager)_

automates some of the work involved in managing a youtube premium family

### features:

- increase/decrease balances with member payments and automated youtube premium charges
- add/remove members to a family
- members can notify the admin about a payment they've made
